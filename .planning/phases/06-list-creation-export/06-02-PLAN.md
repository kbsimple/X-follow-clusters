    Create src/list/exporter.py with export functions for CSV and Parquet.

    Import: pandas as pd, Path, json, logging, from src.review.registry import load_registry
    Import: from src.review.automation import is_automation_enabled

    Create data/export/ directory:

    Function export_clusters_to_csv():
    - Call load_registry() to get approved and deferred clusters
    - For each approved cluster: append row with cluster_id, cluster_name, status="approved", size, silhouette, member_handles (comma-joined usernames), central_member_usernames
    - For each deferred cluster: same fields but status="deferred"
    - Build pandas DataFrame with columns: cluster_id, cluster_name, status, size, silhouette, member_handles, central_member_usernames
    - Call df.to_csv("data/export/clusters.csv", index=False)
    - Log: "Exported {N} clusters to data/export/clusters.csv"
    - Return path to file

    If no clusters exist (empty approved and deferred): create empty CSV with headers only.

    Function export_followers_to_parquet():
    - Scan data/enrichment/ directory for all .json files
    - Skip files with stem in ("suspended", "protected", "errors")
    - For each file: load JSON, append to rows list
    - Build pandas DataFrame from rows
    - Ensure columns include: username, description, location, followers_count, following_count, verified, protected, pinned_tweet_text, cluster_id, cluster_name, silhouette_score, is_seed_category, central_member_usernames, plus any additional fields from scrape
    - Call df.to_parquet("data/export/followers.parquet", index=False)
    - Log: "Exported {N} follower records to data/export/followers.parquet"
    - Return path to file

    If no enrichment files found: raise RuntimeError("No enrichment data found. Run Phase 4 clustering first.")

    Function export_all():
    - Call export_clusters_to_csv()
    - Call export_followers_to_parquet()
    - Print summary using rich: "[green]Export complete[/green] - clusters.csv: {N} rows, followers.parquet: {M} rows"
