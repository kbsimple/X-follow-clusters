    Accounts:

    Group name:'''

    2. def _get_bios_text(central_usernames: list[str], cache_dir: Path) -> str:
       - For each username in central_usernames, load data/enrichment/{username}.json
       - Get description field (bio text)
       - Skip accounts with no description, log a warning
       - Join up to 5 bios with newlines as "- {bio}"
       - Return the formatted string

    3. def name_cluster(bios: list[str], model: str = "gpt-4o-mini") -> str:
       - If len(bios) == 0, return "Unnamed Cluster"
       - If OPENAI_API_KEY not in os.environ and ANTHROPIC_API_KEY not in os.environ:
         Log warning "No LLM API credentials found; using rule-based name"
         Return rule_based_name(bios)
       - If OPENAI_API_KEY present: use OpenAI client with model="gpt-4o-mini", temperature=0.3, max_tokens=32
       - If only ANTHROPIC_API_KEY present: use Anthropic client with model="claude-3-5-haiku-20241017", max_tokens=32
       - Send prompt with bios_text from LLM_NAME_PROMPT
       - Parse response, strip quotes, return name

    4. def rule_based_name(bios: list[str]) -> str:
       - Simple fallback if no LLM available
       - Count occurrences of common keywords: location patterns (capitalized words), tech terms, political terms
       - Return "Interest Group" as default
       - This is a degraded mode, not primary naming

    5. def name_all_clusters(cache_dir: str | Path = Path("data/enrichment"), dry_run: bool = False) -> dict[int, str]:
       - Load all cache files, group by cluster_id
       - For each cluster, get central_member_usernames from cache
       - Load bios for those usernames via _get_bios_text
       - Call name_cluster(bios) to get name
       - If not dry_run: update each member's cache file with cluster_name field
       - If dry_run: only print the names
       - Return dict mapping cluster_id -> cluster_name

    6. On module import, check for API keys and log which provider will be used:
       - If OPENAI_API_KEY: log "Using OpenAI GPT-4o-mini for cluster naming"
       - Elif ANTHROPIC_API_KEY: log "Using Anthropic Claude for cluster naming"
       - Else: log "No LLM credentials found; using rule-based naming" and set a module-level flag
