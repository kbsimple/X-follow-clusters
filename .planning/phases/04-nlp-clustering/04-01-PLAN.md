<task type="auto">
  <name>Task 3: Create embed.py for text embedding and clustering</name>
  <files>src/cluster/embed.py</files>
  <action>
    Create the directory src/cluster/ if it does not exist.

    Create src/cluster/__init__.py exporting cluster_all, ClusterResult, get_text_for_embedding, compute_clusters.

    Create src/cluster/embed.py with:

    1. get_text_for_embedding(account: dict) -> str
       - Per D-01: join description + location + professional_category + pinned_tweet_text with " | " separator
       - Strip whitespace, skip empty parts
       - Return empty string if all fields are empty (account will be skipped)

    2. embed_accounts(accounts: list[dict], model_name: str = "sentence-transformers/all-MiniLM-L6-v2", batch_size: int = 64) -> np.ndarray
       - Load SentenceTransformer(model_name) once
       - Encode texts in batches via model.encode(texts, batch_size=batch_size, normalize_embeddings=True)
       - Show progress bar via show_progress_bar=True
       - Return np.ndarray of shape (n_accounts, 384)
       - Cache embeddings to data/embeddings.npy on first run; load from cache if file exists and n_matches
       - Raise ValueError if fewer than 10 accounts have non-empty text

    3. load_seed_embeddings(seed_accounts: dict[str, list[str]], cache_dir: Path) -> dict[str, np.ndarray]
       - For each category, load each username's cache file from data/enrichment/{username}.json
       - Extract embedding for that account
       - Return dict mapping category -> np.ndarray of shape (n_seeds, 384)

    4. compute_clusters(embeddings: np.ndarray, seed_embeddings_by_category: dict[str, np.ndarray], min_size: int = 5, max_size: int = 50, algorithm: str = "kmeans") -> tuple[np.ndarray, np.ndarray, np.ndarray, list[str]]
       - CLUSTER-02: algorithm parameter supports "kmeans" (default) or "hdbscan"
       - When algorithm="kmeans":
         - Compute centroid for each seed category: np.mean(seed_embeds, axis=0) for each category
         - Stack centroids to create initial_centroids array of shape (n_seed_categories, 384)
         - n_clusters = len(seed_embeddings_by_category) + 3  # seeds + discovered
         - Run: KMeans(n_clusters=n_clusters, init=initial_centroids, n_init=1, random_state=42, algorithm="elkan")
         - Get labels, centroids, inertia
       - When algorithm="hdbscan":
         - Import hdbscan module; if not available, raise ImportError with message "hdbscan not installed"
         - Run: hdbscan.HDBSCAN(min_cluster_size=5, metric='euclidean', prediction_data=True)
         - Get labels (may include -1 for noise/outliers)
         - Compute centroids for each cluster from embeddings of assigned members
         - category_names = list(seed_embeddings_by_category.keys()) + [f"discovered_{i}" for i in range(n_clusters - len(seed_embeddings_by_category))]
       - Post-hoc size enforcement: for any cluster with >max_size (50), reassign farthest members to nearest neighbor cluster. For any cluster with <min_size (5), attempt to absorb from nearest neighbor. If still <5, mark cluster for flagging (do not delete members).
       - Return (labels, centroids, final_centroids, category_names)
       - NOTE: Seed category anchoring is only supported with kmeans; when using hdbscan, seeds are informational only (silhouette guidance)

    5. compute_silhouette_scores(embeddings: np.ndarray, labels: np.ndarray) -> dict[int, float]
       - For each unique label, compute mean silhouette score for members of that cluster using sklearn.metrics.silhouette_samples
       - Return dict mapping cluster_id -> silhouette_score

    6. generate_size_histogram(labels: np.ndarray) -> dict[str, Any]
       - Use np.histogram to create size distribution
       - Return {"counts": list, "bins": list, "total_clusters": n_clusters, "clusters_under_5": count, "pct_under_5": float}

    7. cluster_all(cache_dir: str | Path = Path("data/enrichment"), output_dir: str | Path | None = None, algorithm: str = "kmeans") -> ClusterResult
       - Load all account cache files from cache_dir (exclude suspended.json, protected.json, errors.json)
       - Filter accounts with empty text (skip them, log warning)
       - Build text for each account via get_text_for_embedding
       - Load or compute embeddings via embed_accounts
       - Load seed accounts from config/seed_accounts.yaml
       - Load seed embeddings from cache via load_seed_embeddings
       - Compute clusters via compute_clusters(algorithm=algorithm)
       - Compute silhouette scores via compute_silhouette_scores
       - Generate size histogram
       - For each cluster, identify top-5 most-central members (lowest distance to centroid)
       - Write cluster assignments back to each account's cache file: add cluster_id, cluster_name (temp "Category N"), silhouette_score, is_seed_category, central_member_usernames
       - output_dir defaults to cache_dir (same files updated in place)
       - Return ClusterResult(total_accounts, n_clusters, labels, silhouette_by_cluster, histogram, central_members_by_cluster)

    8. ClusterResult dataclass with: total_accounts, n_clusters, labels, silhouette_by_cluster, size_histogram, central_members_by_cluster
  </action>
