  <name>Task 2: Implement callback server for code capture</name>
  <files>src/auth/x_auth.py</files>
  <action>
    Add a helper function to `src/auth/x_auth.py` that starts a temporary HTTP server to capture the authorization code from the X redirect:

    **wait_for_callback(port: int = 8080, timeout: int = 300) -> str**
    Starts http.server on the given port, waits for a request to `/callback?code=XXX`, extracts the code, and returns it. Raises TimeoutError if no callback received within timeout seconds.

    Implementation approach:
    - Use threading to start http.server in background
    - Monitor for the callback request
    - Extract the `code` query parameter
    - Shut down the server and return the code

    ```python
    import threading
    import time
    from http.server import HTTPServer, BaseHTTPRequestHandler

    def wait_for_callback(port: int = 8080, timeout: int = 300) -> str:
        code_received = threading.Event()
        received_code = [None]

        class CallbackHandler(BaseHTTPRequestHandler):
            def do_GET(self):
                from urllib.parse import urlparse, parse_qs
                if self.path.startswith("/callback"):
                    qs = parse_qs(urlparse(self.path).query)
                    if "code" in qs:
                        received_code[0] = qs["code"][0]
                        code_received.set()
                        self.send_response(200)
                        self.send_header("Content-Type", "text/html")
                        self.end_headers()
                        self.wfile.write(b"<html><body>Authorization complete. You may close this window.</body></html>")
                    else:
                        self.send_response(400)
                        self.wfile.write(b"Missing code parameter")
                else:
                    self.send_response(404)
            def log_message(self, *args): pass  # suppress logging

        server = HTTPServer(("127.0.0.1", port), CallbackHandler)
        thread = threading.Thread(target=server.handle_request)
        thread.start()

        if not code_received.wait(timeout):
            server.shutdown()
            raise TimeoutError(f"No callback received within {timeout} seconds")

        server.shutdown()
        thread.join()
        return received_code[0]
    ```

    **ensure_authenticated() -> XAuth**
    This function orchestrates the full first-run OAuth 2.0 flow:
    1. Try loading tokens from data/tokens.json
    2. If tokens exist, build XAuth from stored tokens + env var client_id/client_secret
    3. If tokens do not exist, start the OAuth 2.0 PKCE flow:
       a. Get authorization URL
       b. Print instructions to user (open URL in browser)
       c. Start callback server
       d. Exchange code for tokens
       e. Save tokens to data/tokens.json
       f. Build XAuth from result
    4. Return XAuth instance

    ```python
    def ensure_authenticated() -> XAuth:
        """Ensure valid OAuth 2.0 tokens are available, running interactive flow if needed."""
        client_id = os.environ.get("X_CLIENT_ID")
        client_secret = os.environ.get("X_CLIENT_SECRET")

        if not client_id or not client_secret:
            raise AuthError("X_CLIENT_ID and X_CLIENT_SECRET must be set in environment")

        # Try loading stored tokens
        tokens = load_tokens()
        if tokens:
            access_token, refresh_token = tokens
            return XAuth(client_id=client_id, client_secret=client_secret,
                        access_token=access_token, refresh_token=refresh_token)

        # First-run: initiate OAuth 2.0 PKCE
        print("No stored tokens found. Initiating OAuth 2.0 PKCE flow...")
        auth_url = get_authorization_url(client_id, client_secret)
        print(f"Open this URL in your browser and authorize:\n{auth_url}")

        code = wait_for_callback()
        access_token, refresh_token = exchange_code_for_token(code)
        save_tokens(access_token, refresh_token)

        return XAuth(client_id=client_id, client_secret=client_secret,
                    access_token=access_token, refresh_token=refresh_token)
    ```
